Each student then gets a score based on the formula

`Score =  Stars × 50 + Commits × 1 + Pull Requests × 3.5 + Repos × 1 + Followers × 1`

Ranks are attributed based on this score. Ties are then broken first with star count, then pr count, then commit count, then follower count, then repo count and finally the username (by lexicographical order).
